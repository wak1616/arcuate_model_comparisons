XGBoost Monotonic Model Parameters
===========================

Parameters:
- objective: reg:squarederror
- max_depth: 4
- learning_rate: 0.05
- subsample: 0.8
- colsample_bytree: 1.0
- reg_alpha: 1.0
- reg_lambda: 5.0
- gamma: 0.1
- monotone_constraints: (0, 0, 0, 0, 1, 0, 0, 0)  # Applies positive monotonicity constraint to 'Treated_astig' (index 4)

Notes:
- The model enforces a positive monotonic relationship for the 'Treated_astig' feature
- This ensures that higher astigmatism values consistently predict larger arcuate sweep values
- All other features remain unconstrained

Model Features:
- Age
- Steep_axis_term
- WTW_IOLMaster
- MeanK_IOLMaster
- Treated_astig (monotonic constraint applied)
- Type (categorical)
- AL
- LASIK? (categorical)
